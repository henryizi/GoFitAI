## üöÄ Quick Hosting Options

### Option 1: GitHub Pages (FREE - Recommended)

**Steps:**
1. **Create a GitHub account** (if you don't have one): https://github.com
2. **Create a new repository:**
   - Go to GitHub ‚Üí New Repository
   - Name it: `gofitai-privacy` (or any name)
   - Make it **Public**
   - Click "Create repository"

3. **Upload the HTML file:**
   - Click "Upload files"
   - Drag and drop `privacy-policy.html`
   - Rename it to `index.html` (important!)
   - Click "Commit changes"

4. **Enable GitHub Pages:**
   - Go to Settings ‚Üí Pages
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click "Save"

5. **Get your URL:**
   - Your privacy policy will be available at:
   - `https://[your-username].github.io/gofitai-privacy/`
   - Or: `https://[your-username].github.io/gofitai-privacy/index.html`

**Example:** `https://henryizi.github.io/gofitai-privacy/`

1. **Upload** `privacy-policy.html` to your web hosting
2. **Rename** to `index.html` in a folder like `/privacy/`
3. **Your URL:** `https://gofitai.com/privacy/`

## üìù Steps to Get Your URL (GitHub Pages)

### Step 1: Create Repository
1. Go to https://github.com/new
2. Repository name: `gofitai-privacy`
3. Make it **Public**
4. Click "Create repository"

### Step 2: Upload File
1. Click "uploading an existing file"
2. Drag `privacy-policy.html` into the upload area
3. **Important:** Rename it to `index.html` before uploading
4. Scroll down, add commit message: "Add privacy policy"
5. Click "Commit changes"

### Step 3: Enable Pages
1. Go to repository **Settings**
2. Click **Pages** in left sidebar
3. Under "Source":
   - Select "Deploy from a branch"
   - Branch: `main`
   - Folder: `/ (root)`
4. Click **Save**

### Step 4: Get Your URL
- Wait 1-2 minutes for deployment
- Your URL will be: `https://[your-username].github.io/gofitai-privacy/`
- Copy this URL for App Store Connect

## üîó Using the URL in App Store Connect

1. **Go to App Store Connect**
2. **Your App ‚Üí App Information**
3. **Scroll to "Privacy Policy URL"**
4. **Paste your URL:**
   ```
   https://[your-username].github.io/gofitai-privacy/
   ```
5. **Save**
