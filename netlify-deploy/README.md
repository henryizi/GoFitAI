# Netlify Deploy Package

This folder contains everything you need to deploy to Netlify.

## Files:
- `_redirects` - Redirect rules for short links
- `index.html` - Landing page (redirects to App Store)

## How to Deploy:

1. Go to [netlify.com](https://netlify.com)
2. Sign up or log in (free)
3. Drag and drop this entire `netlify-deploy` folder onto Netlify
4. Done! You'll get a link like: `https://random-name-123.netlify.app`

## Your Short Links Will Be:

- `your-site.netlify.app/app` → App Store
- `your-site.netlify.app/tiktok` → App Store
- `your-site.netlify.app/ig` → App Store
- `your-site.netlify.app/fb` → App Store
- `your-site.netlify.app/yt` → App Store
- `your-site.netlify.app/download` → App Store

All redirect to: https://apps.apple.com/us/app/gofitai/id6752763510

## Optional: Add Custom Domain

1. In Netlify: Site settings → Domain management
2. Add custom domain (e.g., `go.gofitai.link`)
3. Follow DNS setup instructions
