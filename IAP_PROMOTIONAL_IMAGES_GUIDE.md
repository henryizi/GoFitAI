Each IAP (In-App Purchase) must have a **unique** promotional image. You cannot use the same image for Monthly and Lifetime subscriptions.

## ğŸ’¡ Design Tips

1. **Make them obviously different:**
   - Use completely different color schemes
   - Different layouts
   - Different emphasis

2. **Keep it simple:**
   - Don't overcrowd with text
   - Make key info (price, trial) very large
   - Use clear, readable fonts

3. **Match your brand:**
   - Use GoFitAI colors (orange #FF6B35)
   - Include logo if possible
   - Consistent typography

4. **Test readability:**
   - Text should be readable at small sizes
   - High contrast between text and background
   - Important info stands out
