## 📱 测试前准备
- [ ] 应用已安装到 iPhone 16 Pro
- [ ] 删除旧版本应用（如果有）
- [ ] 确保网络连接正常

## 🎯 核心测试流程

### ✅ 测试 1：付费墙显示
**目标：验证付费墙是否正确显示**

1. [ ] 打开 GoFitAI 应用
2. [ ] 完成注册/登录（如需要）
3. [ ] 完成入门引导
4. [ ] **预期结果：应该看到付费墙界面**
   - [ ] 显示高级功能列表
   - [ ] 有 "Upgrade to Premium" 按钮
   - [ ] 有 "Maybe Later" 按钮

**如果付费墙没有显示：**
- 重新启动应用
- 检查是否跳过了某些步骤

---

### ✅ 测试 2：免费用户限制
**目标：验证免费用户使用限制**

1. [ ] 点击 "Maybe Later" 跳过付费墙
2. [ ] 进入主界面

**测试食谱限制（5个/天）：**
3. [ ] 进入 Nutrition 标签页
4. [ ] 点击 "AI Recipes" 按钮
5. [ ] 生成第1个食谱 ✓
6. [ ] 生成第2个食谱 ✓
7. [ ] 生成第3个食谱 ✓
8. [ ] 生成第4个食谱 ✓
9. [ ] 生成第5个食谱 ✓
10. [ ] 尝试生成第6个食谱 → **应该被阻止并显示付费墙**

**测试AI聊天限制（10条/天）：**
11. [ ] 在 Nutrition 页面点击 "AI Chat" 按钮
12. [ ] 发送1-10条AI聊天消息 ✓
13. [ ] 尝试发送第11条消息 → **应该被阻止并显示付费墙**

---

### ✅ 测试 3：购买流程
**目标：验证购买流程正常工作**

1. [ ] 在付费墙界面点击 "Upgrade to Premium"
2. [ ] **预期结果：打开 RevenueCat/App Store 购买界面**
3. [ ] 使用测试 Apple ID 进行购买
4. [ ] 购买成功后返回应用

---

### ✅ 测试 4：高级功能
**目标：验证购买后功能解锁**

1. [ ] 无限制生成食谱（超过5个）
2. [ ] 无限制AI聊天（超过10条）
3. [ ] 访问所有高级功能

## 🚨 如果测试失败

### 付费墙不显示：
```bash
# 检查应用日志
npx expo logs --platform ios
```

### 限制不生效：
- 检查日期是否正确
- 重新安装应用重置计数器

### 购买流程问题：
- 确认使用测试 Apple ID
- 检查网络连接
- 查看 RevenueCat 配置
