# 📱 Expo Go 测试指南

## 🚀 **快速开始**

### 1️⃣ **启动开发服务器**
```bash
npx expo start
```
✅ **已启动！** 开发服务器正在后台运行

### 2️⃣ **在手机上打开应用**
1. 确保手机和电脑在同一 WiFi 网络
2. 打开 **Expo Go** 应用
3. 扫描终端中显示的 QR 码
4. 等待应用加载完成

---

## 🧪 **付费墙测试步骤**

### ✅ **第一步：跳过初始付费墙**
1. 应用启动后会显示付费墙
2. 点击 **"Maybe Later"** 跳过
3. 进入主界面

### ✅ **第二步：测试 AI 食谱限制**
1. 点击底部 **Nutrition** 标签
2. 找到 **"AI Recipes"** 按钮（橙色图标）
3. 点击进入食谱生成器
4. 生成 5 个食谱 ✅
5. 尝试生成第 6 个 → **应该显示付费墙** 🚫

### ✅ **第三步：测试 AI 聊天限制**
1. 在 Nutrition 页面找到 **"AI Chat"** 按钮（紫色图标）
2. 点击进入 AI 聊天
3. 发送 10 条消息 ✅
4. 尝试发送第 11 条 → **应该显示付费墙** 🚫

### ✅ **第四步：测试购买流程**
1. 当付费墙出现时，点击 **"Upgrade to Premium"**
2. 应该显示购买选项
3. 在 Expo Go 中，这会显示模拟购买界面

---

## 🔧 **如果遇到问题**

### 📱 **应用无法加载：**
```bash
# 重启开发服务器
npx expo start --clear
```

### 🔄 **需要重置使用计数：**
1. 在 Expo Go 中摇晃手机
2. 选择 "Reload" 重新加载应用
3. 或者关闭应用重新扫码

### 🐛 **功能不工作：**
1. 检查终端是否有错误信息
2. 确保网络连接正常
3. 尝试重新加载应用
